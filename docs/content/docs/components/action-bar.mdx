## Layout

```tsx
import {
  ActionBar,
  ActionBarClose,
  ActionBarItem,
  ActionBarSelection,
  ActionBarSeparator,
} from "@/components/ui/action-bar";

<ActionBar open={open} onOpenChange={setOpen}>
  <ActionBarSelection>
    {count} selected
    <ActionBarClose />
  </ActionBarSelection>
  <ActionBarSeparator />
  <ActionBarItem>Action 1</ActionBarItem>
  <ActionBarItem>Action 2</ActionBarItem>
</ActionBar>
```

## Accessibility

The action bar follows WAI-ARIA patterns for dialog components:

- Uses proper `role="separator"` for separators
- Interactive items are keyboard accessible
- Focus management is handled automatically
- Supports both controlled and uncontrolled states

## Notes

- The action bar is rendered in a portal to ensure proper stacking context
- Animations can be disabled by setting `animated={false}`
- Position can be customized using the `side`, `align`, and `sideOffset` props
- The component automatically handles positioning based on viewport alignment
