### Group

A container for action items that implements roving focus management. Items within a group can be navigated using arrow keys, forming a single tab stop. Has `role="group"` for accessibility.

**Keyboard Navigation:**
- `ArrowLeft` / `ArrowRight` (horizontal) or `ArrowUp` / `ArrowDown` (vertical): Navigate between items
- `Home`: Focus first item in group
- `End`: Focus last item in group
- `Tab`: Exit the group and move to next focusable element
