## Layout

Import the parts, and compose them together.

```tsx
import * as SpeedDial from "@/components/ui/speed-dial";

return (
  <SpeedDial.Root>
    <SpeedDial.Trigger />
    <SpeedDial.Content>
      <SpeedDial.Item>
        <SpeedDial.Label />
        <SpeedDial.Action />
      </SpeedDial.Item>
    </SpeedDial.Content>
  </SpeedDial.Root>
)
```
