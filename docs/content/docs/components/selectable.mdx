## Layout

Import the parts, and compose them together.

```tsx
import { Selectable, SelectableItem } from "@/components/ui/selectable"

<Selectable>
  <SelectableItem value="option-1">Option 1</SelectableItem>
  <SelectableItem value="option-2">Option 2</SelectableItem>
  <SelectableItem value="option-3">Option 3</SelectableItem>
</Selectable>
```

## Examples

### Basic Usage

The selectable component can be used to create a keyboard-navigable list of items.

<ComponentTabs name="selectable-demo" />

### Horizontal Orientation

Set `orientation="horizontal"` to create a horizontally navigable list.

<ComponentTabs name="selectable-horizontal-demo" />

### Grid Layout

For grid layouts, use `orientation="mixed"` to enable navigation in both directions. 
Use CSS Grid to arrange the items in a grid structure. In grid layouts, 
arrow keys will navigate accordingly:

- Up/Down: Navigates within a column
- Left/Right: Navigates within a row
