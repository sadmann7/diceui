## Installation

### CLI

```package-install
npx shadcn@latest add "https://diceui.com/r/kanban"
```

### Manual

## Layout

Import the parts, and compose them together.

```tsx
import * as Kanban from "@/components/ui/kanban";

<Kanban.Root>
  <Kanban.Board>
    <Kanban.Column>
      <Kanban.ColumnHandle />
      <Kanban.Item>
        <Kanban.ItemHandle />
      </Kanban.Item>
    </Kanban.Column>
  </Kanban.Board>
  <Kanban.Overlay />
</Kanban.Root>
```
