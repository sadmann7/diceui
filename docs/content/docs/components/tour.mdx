## Layout

Import the parts, and compose them together.

```tsx
import * as Tour from "@/components/ui/tour";

<Tour.Root>
  <Tour.Overlay />
  <Tour.Content>
    <Tour.Close />
    <Tour.Header>
      <Tour.Title />
      <Tour.Description />
    </Tour.Header>
    <Tour.Footer>
      <Tour.StepCounter />
        <Tour.Prev />
        <Tour.Next />
      <Tour.Skip />
    </Tour.Footer>
  </Tour.Content>
</Tour.Root>
```
