### ARIA Attributes

- Uses `role="radiogroup"` for the rating container
- Each rating item has `role="radio"` 
- Proper `aria-label` and `aria-describedby` attributes for screen readers
- `aria-setsize` and `aria-posinset` for position information
- `aria-checked` state for selected items

## Data Attributes

### Root

- `data-disabled` - Present when the rating is disabled
- `data-readonly` - Present when the rating is read-only
- `data-invalid` - Present when the rating is in an invalid state

### Item

- `data-state` - The current state: "active", "inactive", or "half"
- `data-disabled` - Present when the item is disabled
- `data-readonly` - Present when the item is read-only
