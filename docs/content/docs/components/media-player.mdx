## Installation

### CLI

```package-install
npx shadcn@latest add "https://diceui.com/r/media-player"
```

## Layout

Import the parts, and compose them together.

```tsx
import * as MediaPlayer from "@/components/ui/media-player";

<MediaPlayer.Root>
  <MediaPlayer.Video>
    <source src="..." type="video/mp4" />
  </MediaPlayer.Video>
  <MediaPlayer.Overlay />
  <MediaPlayer.Loading />
  <MediaPlayer.Controls>
    <MediaPlayer.Play />
    <MediaPlayer.Volume />
    <MediaPlayer.Seek />
    <MediaPlayer.Time />
    <MediaPlayer.PlaybackSpeed />
    <MediaPlayer.Loop />
    <MediaPlayer.Captions />
    <MediaPlayer.Resolution />
    <MediaPlayer.PiP />
    <MediaPlayer.Fullscreen />
    <MediaPlayer.Download />
  </MediaPlayer.Controls>
</MediaPlayer.Root>
```

### Live Streaming Player

The media player includes support for live streaming with special controls and indicators:

```tsx
import * as MediaPlayer from "@/components/ui/media-player";

export default function LivePlayer() {
  return (
    <MediaPlayer.Root isLive className="w-full max-w-4xl">
      <MediaPlayer.Video
        src="https://your-live-stream-url.m3u8"
        poster="/stream-poster.jpg"
      />
      <MediaPlayer.Overlay />
      <MediaPlayer.Controls>
        <MediaPlayer.Play />
        <MediaPlayer.Live />
        <MediaPlayer.Volume />
        <div className="flex-1" />
        <MediaPlayer.Settings />
        <MediaPlayer.Cast />
        <MediaPlayer.AirPlay />
        <MediaPlayer.Fullscreen />
      </MediaPlayer.Controls>
    </MediaPlayer.Root>
  );
}
```

### Resolution

A dropdown menu to select video quality/resolution for videos with multiple renditions (HLS/DASH).

<AutoTypeTable
  path="./types/docs/media-player.ts"
  name="ResolutionProps"
/>

**Features:**
- Only appears when multiple video renditions are available
- Shows current quality in button (e.g., "1080p", "720p")
- Includes "Auto" option for automatic quality selection
- Sorts resolutions by quality (highest first)
- Uses `mediaRenditionList` from media-chrome

**Supported formats:**
- HLS (.m3u8) streams
- DASH (.mpd) streams
- Custom media elements with rendition support
