## Layout

Import the parts and compose them together.

```tsx
import {
  AngleSlider,
  AngleSliderRange,
  AngleSliderThumb,
  AngleSliderTrack,
  AngleSliderValue,
} from "@/components/ui/angle-slider";

<AngleSlider>
  <AngleSliderTrack>
    <AngleSliderRange />
  </AngleSliderTrack>
  <AngleSliderThumb />
  <AngleSliderValue />
</AngleSlider>
```

### Features

- **Touch Support**: Optimized for touch interactions on mobile devices
- **Pointer Capture**: Smooth dragging experience with proper pointer handling
- **RTL Support**: Full right-to-left language support
- **High Contrast**: Respects user's high contrast preferences
- **Reduced Motion**: Honors `prefers-reduced-motion` settings
