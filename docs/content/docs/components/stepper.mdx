## Layout

Import the parts, and compose them together.

```tsx
import * as Stepper from "@/components/ui/stepper";

<Stepper.Root>
  <Stepper.List>
    <Stepper.Item >
      <Stepper.Trigger>
        <Stepper.Indicator />
      </Stepper.Trigger>
      <Stepper.Title />
      <Stepper.Description />
      <Stepper.Separator />
    </Stepper.Item>
  </Stepper.List>
  <Stepper.Content />
</Stepper.Root>
```
