## Layout

Import the parts, and compose them together.

```tsx

return (
  <TimePicker.Root>
    <TimePicker.Label />
    <TimePicker.Trigger />
```

## Notes

- The time value is formatted as "HH:mm" or "HH:mm:ss" in 24-hour format internally
- When using `use12Hours`, the value is still stored in 24-hour format for consistency
- The `minuteStep`, `hourStep`, and `secondStep` props allow customization of selection intervals
- Use the `min` and `max` props to restrict the selectable time range
- The component automatically handles conversion between 12-hour and 24-hour formats
