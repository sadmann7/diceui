## Layout

Import the parts, and compose them together.

```tsx
import * as ScrollSpy from "@/components/ui/scroll-spy"

return (
  <ScrollSpy.Root>
    <ScrollSpy.ItemGroup>
      <ScrollSpy.Item />
    </ScrollSpy.ItemGroup>
    <ScrollSpy.ContentGroup>
      <ScrollSpy.Content />
    </ScrollSpy.ContentGroup>
  </ScrollSpy.Root>
)
```

## Examples

### Sticky Layout

For full-page scroll behavior, you can use a sticky positioned navigation sidebar that stays fixed while the content scrolls. This works with the default window scroll (no `scrollContainer` prop needed).

```tsx
<ScrollSpy offset={100}>
  <ScrollSpyItemGroup className="sticky top-20 h-fit">
    <ScrollSpyItem value="introduction">Introduction</ScrollSpyItem>
    <ScrollSpyItem value="getting-started">Getting Started</ScrollSpyItem>
    <ScrollSpyItem value="usage">Usage</ScrollSpyItem>
    <ScrollSpyItem value="api-reference">API Reference</ScrollSpyItem>
  </ScrollSpyItemGroup>

  <ScrollSpyContentGroup>
    <ScrollSpyContent value="introduction">
      <h2>Introduction</h2>
      <p>Your content here...</p>
    </ScrollSpyContent>
    
    <ScrollSpyContent value="getting-started">
      <h2>Getting Started</h2>
      <p>Your content here...</p>
    </ScrollSpyContent>
    
    {/* More content sections */}
  </ScrollSpyContentGroup>
</ScrollSpy>
```

The key is to apply `sticky top-[offset]` to the `ScrollSpyItemGroup` to keep the navigation visible as the page scrolls.
