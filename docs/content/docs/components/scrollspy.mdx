## Layout

Import the parts, and compose them together.

```tsx
import * as ScrollSpy from "@/components/ui/scrollspy"

return (
  <ScrollSpy.Root>
    <ScrollSpy.List>
      <ScrollSpy.Item />
    </ScrollSpy.List>
    <ScrollSpy.Content />
  </ScrollSpy.Root>
)
```
