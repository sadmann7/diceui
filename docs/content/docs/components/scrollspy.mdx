## Layout

Import the parts, and compose them together.

```tsx
import * as ScrollSpy from "@/components/ui/scrollspy"

<ScrollSpy.Root>
  <ScrollSpy.List>
    <ScrollSpy.Item value="section-1">Section 1</ScrollSpy.Item>
    <ScrollSpy.Item value="section-2">Section 2</ScrollSpy.Item>
  </ScrollSpy.List>

  <ScrollSpy.Content id="section-1">
    {/* Section 1 content */}
  </ScrollSpy.Content>

  <ScrollSpy.Content id="section-2">
    {/* Section 2 content */}
  </ScrollSpy.Content>
</ScrollSpy.Root>
```
