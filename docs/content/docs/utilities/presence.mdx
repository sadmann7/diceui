## Usage

```tsx
import { Presence } from "@/components/presence"

export default function App() {
  const [isOpen, setIsOpen] = React.useState(false)

  return (
    <Presence present={isOpen}>
      <div className="animate-in fade-in">
        This content will animate in and out
      </div>
    </Presence>
  )
}
```

You can also use it with a function as children to access the present state:

```tsx
import { Presence } from "@/components/presence"

export default function App() {
  const [isOpen, setIsOpen] = React.useState(false)

  return (
    <Presence present={isOpen}>
      {({ present }) => (
        <div className={present ? "animate-in fade-in" : "animate-out fade-out"}>
          This content will animate based on presence state
        </div>
      )}
    </Presence>
  )
}
```

## API Reference

### Presence

A component that manages the presence state of elements with support for animations. It handles mounting, unmounting, and animation states automatically.

#### Props

<AutoTypeTable
  path="./types/docs/utilities.ts"
  name="PresenceProps"
/>

The component supports the following props:

- `present` - Boolean indicating whether the element should be present in the DOM
- `children` - React element or render function that receives the present state
- `forceMount` - Optional boolean to force mounting regardless of present state

#### Features

- Manages element presence with animation support
- Handles animation and transition events automatically
- Supports both CSS animations and transitions
- Maintains animation states correctly during unmounting
- Provides present state to child components through render props
